Este proyecto consiste en el desarrollo de una aplicación para la gestión de bibliotecas, utilizando Angular para el frontend y .NET para el backend. La aplicación permite gestionar libros, usuarios y préstamos de manera eficiente, ofreciendo una interfaz moderna y funcional.

Gestión de libros y usuarios: Agregar, editar y eliminar libros y usuarios de la biblioteca.
Gestión de préstamos: Registrar préstamos y devoluciones de libros, con control de fechas y disponibilidad.
Búsqueda y filtrado: Los usuarios pueden buscar libros por título, autor o categoría.
Seguridad: Autenticación y autorización básica para asegurar que solo usuarios registrados puedan realizar ciertas acciones.

Tecnologías Utilizadas:

Frontend: Angular, Material, HTML, SCSS

Backend: .NET Core, C#

Base de Datos: SQL Server

Otros: Entity Framework, REST API, JSON

Arquitectura: El proyecto sigue una arquitectura por capas, facilitando la escalabilidad y el mantenimiento del sistema.

Pruebas y Calidad: Se implementaron pruebas unitarias con xUnit en el backend para asegurar el correcto funcionamiento de la lógica de negocio.
